# SCT_WD_2 - Stopwatch Web Application

## 📌 Project Overview
This project is a simple and interactive stopwatch web application developed as part of the SkillCraft Technology Web Development Internship (Task 2). It allows users to accurately measure time with start, stop, and reset functionalities.

## 🚀 Features
- Start, Stop, and Reset stopwatch
- Accurate time tracking (minutes, seconds, milliseconds)
- Responsive and clean UI design
- Smooth user interactions

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Objective
To build a functional stopwatch application while improving skills in JavaScript DOM manipulation, event handling, and time-based functions.
